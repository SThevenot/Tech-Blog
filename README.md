# Tech Blog Application 
  ![](https://img.shields.io/badge/license--green)

  #### The goal of this application is to create a fully functional tech blog website. It allows you to sign up and log in, and only then can you create posts. Using routes, the user is able to create, edit, and delete posts as they wish and timestamps are also attached to every post created. The user can also view the previous posts and log out if they wish.
